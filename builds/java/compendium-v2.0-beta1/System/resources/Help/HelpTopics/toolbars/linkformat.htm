The Link Format Toolbar

This toolbar is only available for Map views.

Figure 1. The Link Format Toolbar


Toolbar option

Allows you to
  Link Colour set the link colour for the selected links.

Link Weight

select the link width for the selected links.
Link Arrows select the arrow head style for the selected links
Link Type select the link style, Straight Link, Curved Link or Squared Link, for the selected links
Link Style select the link line style: Plain Line; Large Dashes; Small Dashes, for the selected links
