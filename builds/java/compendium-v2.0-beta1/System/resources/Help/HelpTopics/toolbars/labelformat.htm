The Label Format Toolbar

This toolbar is only available for Map views at present.

Figure 1. The Label Format Toolbar


Toolbar option

Allows you to

Font Face Menu

select the font face to apply to the selected node/link labels.

Font Size Menu set the font size to apply to the selected node/link labels
  Bold make all selected node/link labels text bold
  Italic make all selected node/link labels text italic
  Foreground Colour set the text colour for the selected node/link labels.
  Background Colour set the label background colour for the selected node/link labels.
 Label Wrap width menu set the label wrap width on the selected node/link labels. This figure is the number of character that can be typed before the label should wrap down to the next line. Label wrapping tried to wrap on whole words, so this figure is only a guide.
