Supported Movie Formats

Compendium can support the playing of Access Grid Streams.
Compendium uses the Java Media Framework (JMF) as a base for playing movies and it also uses Fobs4JMF library which provides access to the FFMPEG library and it's codecs.
FFMPEG provides Compendium the ability to play many movie formats.

N.B. The Compendium Team would like to thank Andrew Rowley and his team at Manchester University for the libraries they wrote to enable Compendium to:

  1. support Access Grid Streams
  2. support most current movie types and codecs by enabling Compendium to connect to the Fobs4JMF library and thus FFMPeg.

JMF 2.1.1 - Supported Media Formats

JMF supports audio sample rates from 8KHz to 48KHz. Note that cross-platform version of JMF only supports the following rates: 8, 11.025, 11.127, 16, 22.05, 22.254, 32, 44.1, and 48 KHz.

The JMF 2.1.1 Reference Implementation supports the media types and formats listed in the table below. In this table:

  • D indicates the format can be decoded and presented.
  • E indicates the media stream can be encoded in the format.
  • read indicates the media type can be used as input (read from a file)
  • write indicates the media type can be generated as output (written to a file)
