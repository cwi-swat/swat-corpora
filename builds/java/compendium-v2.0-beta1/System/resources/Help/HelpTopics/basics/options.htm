User Options (Preferences)


The user options (user preferences on the Mac), dialog can be located on the Tools Menu.

The 'DnD Folders' tab, as seen in figure 2., has the drag and drop options on it for folder drops.  The following table lists them with explanations:


Option

Purpose

Drop folders as: link to original folder

if you select this option, then when you drag a folder into Compendium it will create a reference node linking to the folder.

Drop folders as: map node

If you select this option, when you drag a folder into compendium it will create a map node representing the folder and nodes inside for the files in that folder (one level deep only).

Drop folders as: map node recursively

If you select this option then when you drag a folder into Compendium it will create a map node representing that folder and nodes inside for the files in that folder. It will create new maps nodes for folders and thier contents recursively.

Drop folders as: always prompt

 If you select this option, you will always be asked when you drop a folder onto Compendium how you wish to process it.

The 'Map & Rollover' tab currently has three options, (see figure 3):

Figure 3. The Map & Rollover tab.


Option

Purpose

Cursor Movement Distance

Allows you to set the distance the cursor is moved when using the arrow keys to move it around.

Scale oversized images on rollover?

This option applies when you have selected to display images on rollover. If you have selected to display images on rollover they would normally be displayed in their actual size. If you wish large images to be scaled to fit the visible area of the view, then check this option.
Detail rollover length Allows you to change the amount of detail text that is displayed when you rollover a '*', on a node.

The default cut off point is 250 characters. This option allows you to enter an different length.
Enable single click for opening nodes? If you select this option then, when in a map, all map and list nodes with be opened with a single click. All reference nodes will be launched with a single click and all other nodes will have their contents dialog opened with a single click instead of a double click.

 

The 'Audio & Zoom' tab has options to set the audio and zoom, (figure 4):

Figure 4. The Audio & Zoom tab

The first one consists of two radio buttons which allow you to choose to have audio sounds in Compendium, for example when you delete a node you could here a sound etc.

The second options is a dropped down menu which allows you to set the default zoom level that views open at.

The 'Arrange' tab currently has four options, (see figure 5):

Figure 5. The Arrange tab.


Option

Purpose

Left to Right: Vertical Gap

Sets the vertical gap between nodes when doing a 'Left to Right' arrange.

Default = 20

Left to Right: Horizontal Gap

Sets the horizontal gap between nodes when doing a 'Left to Right' arrange.

Default = 30

Top-Down: Vertical Gap

Sets the vertical gap between nodes when doing a 'Top-Down' arrange.

Default = 40
Top-Down: Horizontal Gap

Sets the horizontal gap between nodes when doing a 'Top-Down' arrange.

The 'Misc' tab currently has one option, (see figure 6):

Figure 6. The Misc tab.

Option

Purpose

Iocn Sets

This allows you to change the icon set that is used for the main node types.

Look and Feel

This allows you to switch look and feel.

N.B. Sometimes you need to restart Compendium for a clean repaint of the look and feel.

Enabled advanced user interface?

If you check this option all menus and toolbars will remain perminantly expanded rather than requiring extending to see full option range.

Email when item goes into Inbox?

If you check this option then when someone sends a node to you inbox you will be emailed.

Use kfmclient to open files?

This option only applies to Linux users. If you are using the KDE desktop then you can choose to use kmfclient, the KDE tool for opening URLs from the command line, to open references nodes.

the advantage is that it does automatic file type recognition so you so you don't need to tell Compendium which application to open a file in ever.
