Linked Files Browser

The Linked Files Browser is a window that allows you to managed your linked files.


Figure 1. The Linked Files browser window

In figure 1., you can see that the Linked Files browser has three columns.

  1. Location: This shows you where the linked file is. It either says 'Database' if the file is held in the database, or it shows you the path to the file. If there is a reference to an image that is no loger found at the path given, it will have 'LOST:' prepended to the path.
  2. File name: This shows you the name of the linked file.
  3. Times used: This shows you how many nodes are referencing the linked file.

If you want to see which nodes are referencing a given linked file, simply select the file row and either double-click it, or press the 'Enter' key. This will open a second window similar to figure 2 below:


Figure 1. The Linked Files browser node usage window

If you want to see more details about any node in the Usage window, simple double-click it, or press the 'Enter' key with it selected and it will open that node's content dialog.

Deleting

To delete one or more files, select them and then press the 'Delete' key. You will be asked for confirmation.
NOTE: You can only delete files that are not LOST and that have no node usage.

Extracting

To get files out of the database, or copy files to a new location, select the file(s) and then press 'Extract..'. The file browser window will open asking you for the location to extract the file(s) to. Navigate to where you want the file(s) to go and then select 'open'. A copy of the file(s) will be placed there.
